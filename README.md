# Chess-game-S4 ♟️

Projet réalisé dans le cadre de la formation **IMAC**.

<u> **Matières</u> :** Mathématiques • Programmation Objet • Synthèse d'Images

---

## 📝 Présentation

Jeu d'échecs complet en **2D et 3D**, avec interface graphique *ImGui* et rendu *OpenGL*.![alt text](image-1.png)

### 🎮 Modes de jeu
![alt text](image.png)
- **Joueur vs Joueur** — deux joueurs sur le même écran
- **Joueur vs IA** — affrontez une intelligence artificielle
- **Mode Infernal** — mode aléatoire

### 🕹️ Comment jouer ?

- **Cliquez sur une pièce** pour la sélectionner — les coups valides s'affichent
- **Cliquez sur une case en surbrillance** pour jouer le coup
- Les vues **2D et 3D** sont affichées simultanément — on peut jouer depuis l'une ou l'autre
- Un **historique des coups** joués est affiché en temps réel
- Il est possible  **d'annuler le dernier coup** à tout moment


### ⌨️ Contrôles

| Touche / Action | Effet |
|---|---|
| **Clic gauche** | Sélectionner / déplacer une pièce |
| **Echap** | Pause |
| **P** | Basculer en Vue Pièce (caméra à la hauteur de la pièce sélectionnée) |
| **Souris (clic gauche + glisser)** | Faire pivoter la caméra autour du plateau |
| **Souris (clic droit + glisser)** | Déplacer la caméra |
| **Molette** | Zoom |
